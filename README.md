# 🎮 Focus Manager RPG – Zaman Yönetimini Oyunlaştır!

Zamanı yönetmek artık sıkıcı değil!  
**Focus Manager RPG**, görev yönetimini eğlenceli bir RPG deneyimine dönüştüren JavaFX tabanlı masaüstü uygulamadır.

## 🚀 Özellikler
- 👤 Kadın / Erkek karakter seçimi (görsel desteğiyle)
- 🧠 Görev tanımlama: Ders adı, amaç ve hatırlatma saati
- 🔔 Hatırlatıcı: Seçilen saatte kullanıcıya bildirim
- ⚔️ XP sistemi: Görev tamamladıkça deneyim puanı kazan
- 🧱 Seviye & Rütbe: Acemi → Çalışkan → Usta
- 🧩 Genişletilebilir yapı: İleride veritabanı ve kayıt sistemi eklenebilir

## 🖼️ Ekran Görüntüsü

## 🛠️ Kullanılan Teknolojiler
- Java 11+
- JavaFX (UI framework)
- Timer API (Hatırlatıcı için canlı zaman kontrolü)

## 🔧 Nasıl Çalıştırılır?
1. JavaFX destekli bir IDE (IntelliJ, Eclipse) açın
2. `FocusManagerRPG.java` dosyasını ekleyin
3. Çalıştırın – Start tuşuna gerek kalmadan direkt başlar
